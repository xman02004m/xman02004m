## 🛡️ Kavach — AI-Powered Threat Detection System

> *My flagship project. Built from scratch. This one's personal.*

**Kavach** (meaning *Shield* in Hindi) is an AI firewall and honeypot-based network threat detection system designed to detect, log, and respond to intrusions in real time.

### What it does
- Deploys a **honeypot** to lure and fingerprint attackers
- Uses **ML-based anomaly detection** to flag suspicious traffic patterns
- Logs all threat events with timestamps, source IPs, and attack types
- Sends real-time alerts when a threat is detected

### Why I built it
Most threat detection tools are expensive, enterprise-only, or black boxes. Kavach is an attempt to build something lightweight, explainable, and deployable even on a Raspberry Pi — making real security accessible.

### Tech Stack
`Python` · `Scapy` · `Scikit-learn` · `Raspberry Pi` · `Linux` · `SQLite`
